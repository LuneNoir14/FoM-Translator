==================================================
FIELDS OF MISTRIA TRANSLATOR APP
==================================================

Hello! This application is an easy and fast 
translation tool developed for the Fields of Mistria game.

HOW TO USE
--------------------------------------------------
1. Run "FoMTranslator.exe".
2. Click the "Open Project" button in the top right.
3. Select the folder containing the game's 
   "localization.en.json" file.
4. On the left side, you'll see categories and entries.
   You can click the "Untranslated" tab to focus only 
   on the strings that still need translation.
5. Write your translation in the text box on the right 
   (your progress auto-saves automatically).
6. When you are done, click the "Export JSON" button 
   in the top right to save the newly translated 
   JSON file to bring back into the game or a mod.

EXTRA FEATURES
--------------------------------------------------
* "Next Untranslated" button rapidly jumps to the 
  next unfinished entry for fast workflow.
* Use the "Search key or text" bar to instantly find 
  any specific dialog or item name in the project.
* Make sure to preserve any special formatting or 
  color codes (e.g., <color="red">) used by the game 
  when you are typing your translations.

Happy translating! 🐔

--------------------------------------------------
TROUBLESHOOTING (SMARTSCREEN WARNING)
--------------------------------------------------
As this application is an open-source, indie project 
without a commercial Microsoft Code Signing Certificate, 
you might encounter a blue "Windows protected your PC" 
warning from Windows SmartScreen upon first launch. 
This is completely normal for unsigned apps.

1. Click "More Info" on the screen.
2. Click "Run Anyway".

You will only need to do this once. The app is 100% 
self-contained and safe. Your exported JSON files 
will default to saving on your Desktop for easy access.
