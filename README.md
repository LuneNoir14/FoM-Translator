# FoMTranslator

Electron + React tabanli yerel ceviri editoru prototipi.

## Komutlar

- `npm.cmd run dev`
- `npm.cmd run test`
- `npm.cmd run build`
- `npm.cmd run dist`

## Notlar

- Testler `better-sqlite3` modülünü mevcut Node surumuyle calistirir.
- Paketleme adimi Electron ABI icin `better-sqlite3` yeniden derler.
- Hedef kaynak dosya: `C:\Users\ANIL\Desktop\çeviri\localization.en.json`

## Mevcut Ozellikler

- Dense dark-theme workbench shell
- Entry listesi, filtre ve secim akisi
- Kaynak metin gosterimi
- Turkce ceviri editoru
- Placeholder ve line-break validation tabani
- JSON export servisi
